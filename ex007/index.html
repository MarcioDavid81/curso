Principais Formatações de Texto

Negrito/Destaque

Neste texto, temos um trecho em negrito. (estilo)

Neste texto, temos um trecho em destaque. (semântica)

Itálico/Ênfase

Neste texto, temos um texto em itálico.(estilo)

Neste texto temos um texto em ênfase. (semântica)

Texto Marcado

Podemos tambem criar um texto marcado usando a tag MARK. E alteramos a cor da marcação usando a tag style. Lembrando que style é uma tag de CSS, portanto temos, neste caso, uma amostra de uso de CSS in line.

Texte Deletado e Texto Inserido

Neste parágrafo temos um texto deletado para indicar que ele deve ser lido mas não considerado

Neste parágrafo temos um texto inserido para indicar que ele foi adicionado posteriormente.

Existe tambem a opção de texto sublinhado com o uso da tag <u> que não deve ser usada, pois trata-se de uma tag não semântica, ou seja, uma tag de estilo. E para isso devemos usar o CSS.

Texto Sobrescrito e Texto Subscrito

Texto sobrescrito serve para inserir caracteres do tipo: 2 elevado a 20, que fica: 220.

Texto subescrito serve para inserir caracteres do tipo H2O, que fica: H2O.
